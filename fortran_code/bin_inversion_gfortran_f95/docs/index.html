Kinematic modelling

This code allows to solve a nonlinear finite fault inverse problem using simmulated annealing. The inversion is performed in wavelet domain.

This version of the code is written in Fortran95, and has been parallelized using OPENMP.

Note

This code is modified from the original, written by Dr. Chen-Ji. We claim no intellectual property whatsoever over this code.

Bug

We have observed, this code doesn't work well when compiled with ifort and OPENMP. In order to compile with ifort, we suggest removing the OPENMP compilation options.
